## Description

Site vitrine développé bénévolement pour l'association **Saves The Soul**. Le projet présente l'association, ses événements, et permet aux visiteurs de contribuer via des dons ou l'achat de billets.

**Site en ligne** : [savesthesoul.fr](https://savesthesoul.fr)

---

## Objectifs du projet

- Créer une vitrine web professionnelle pour promouvoir l'association
- Permettre la collecte de dons, et billeterie via HelloAsso
- Présenter les événements organisés
- Offrir un point de contact direct avec les visiteurs

---

## Fonctionnalités

### Page d'accueil
- Présentation de l'association et de sa mission
- Présentation des fondateurs avec liens vidéos
- Formulaire de contact avec protection anti-spam
- Intégration HelloAsso pour les dons

### Page Événement
- Informations complètes sur l'événement (date, lieu, horaires, prix)
- Présentation du thème et du programme détaillé
- Billetterie en ligne (via HelloAsso)
- Intégration HelloAsso pour les dons

### Page Boutique *(développée mais non publiée)*
- Catalogue de produits avec pages détaillées
- Interface responsive pour la navigation produits
- Fiches produits individuelles

### Pages légales
- Mentions légales
- Politique de confidentialité
- Conditions Générales d'Utilisation

---

## Technologies utilisées

- **HTML5** : Structure sémantique et accessible
- **CSS3** : Mise en page responsive, animations et effets visuels
- **JavaScript** : Menu déroulant et interactions (fourni)
- **PHP** : Traitement du formulaire de contact (fourni)

## Installation & Déploiement

### Prérequis
- Serveur web (Apache, Nginx, etc.)
- PHP 7.4+ (pour le formulaire de contact)

### Installation locale
```bash
# Cloner le repository
git clone https://github.com/[ton-username]/saves-the-soul.git

# Ouvrir le fichier index.html dans un navigateur
# Ou utiliser un serveur local (ex: Live Server sur VS Code)
```

### Structure des fichiers
```
saves-the-soul/
├── index.html              # Page d'accueil
├── evenement.html          # Page événement
├── Boutique.html           # Page boutique (non publiée)
├── produit/                # Pages produits individuels (non publiée)
├── style.css               # Feuille de styles principale
├── script.js               # Scripts JavaScript
├── contact.php             # Traitement formulaire
└── reglementation/         # Pages légales 
    ├── mentionslegales.html
    ├── politiqueconfi.html
    └── cgu.html
```
